# Biblioteca Doce Leitura - Sistema Web

## 📖 Sobre o Projeto

Este projeto é a versão Web do sistema originalmente desenvolvido em Java Desktop.  
A aplicação foi reestruturada para arquitetura Web utilizando Spring Boot e Api Rest, aplicando princípios SOLID e padrão MVC.

O sistema permite:  
- Cadastro  
- Listagem  
- Atualização  
- Exclusão  

---

## 🚀 Tecnologias Utilizadas

- Java 17+
- MySQL
- NetBeans
- Spring Boot
- Maven
- Git e GitHub
